# 前端参考

InSARHub Web 界面每个面板的详细参考。

## 下载器面板

点击任务文件夹上的**下载器**标签打开。

### 配置选项卡
显示存储在 `insarhub_config.json` 中的下载器配置。点击任意行复制值。

### 操作

| 按钮 | 说明 |
|--------|-------------|
| **查看网络** | 显示干涉图网络图 |
| **查看配对** | 列出所有选定的干涉图配对 |
| **选择配对** | 打开配对选择对话框 — 配置 `dt_targets`、容差和度数限制 |
| **下载** | 将所有可用的 HyP3 干涉图下载到磁盘 |
| **提交** | 将选定配对提交至处理器（HyP3） |

### 选择配对对话框

| 字段 | 说明 |
|-------|-------------|
| **时间基线目标** | 以逗号分隔的目标时间基线（天），例如 `12, 24, 48` |
| **时间基线容差** | 与各目标的允许偏差（天） |
| **最大时间基线** | 最大时间基线（天） |
| **最小 / 最大度数** | 网络中每个场景的最小和最大连接数 |
| **强制连通** | 确保网络中无孤立场景 |

## 处理器面板

点击任务文件夹上的**处理器**标签打开。

### HyP3 任务文件

列出文件夹中的所有 `hyp3_jobs.json` 文件。选择一个进行操作。

### 操作

| 按钮 | 说明 |
|--------|-------------|
| **刷新** | 向 HyP3 轮询当前任务状态 |
| **重试** | 重新提交所有失败的任务 |
| **下载** | 下载所有成功的干涉图 |
| **停止** | 取消正在进行的下载 |

---

## 分析器面板

点击任务文件夹上的**分析器**标签打开。

### 步骤清单

每个 MintPy 处理步骤均显示复选框，选择要运行的步骤：

| 步骤 | 说明 |
|------|-------------|
| `prep_data` | 准备并裁剪 HyP3 产品，写入 `mintpy.cfg` |
| `load_data` | 将干涉图加载到 MintPy HDF5 堆栈 |
| `modify_network` | 删除低相干性干涉图 |
| `reference_point` | 选择稳定参考像素 |
| `quick_overview` | 生成诊断概览图层（相干性、相位速度、解缠误差、连通分量掩模）— 启用**概览**按钮 |
| `invert_network` | SBAS 反演 → 时序 |
| `correct_LOD` | 校正本地振荡器漂移 |
| `correct_SET` | 固体地球潮汐校正 |
| `correct_troposphere` | 对流层延迟校正 |
| `deramp` | 去除残余斜坡信号 |
| `correct_topography` | DEM 误差校正 |
| `residual_RMS` | 计算残差 RMS |
| `reference_date` | 设置参考日期 |
| `velocity` | 估算线性速度图 |
| `geocode` | 将结果地理编码到地理坐标 |
| `hdfeos5` | 导出为 HDF-EOS5 格式 |

### 按钮

| 按钮 | 说明 |
|--------|-------------|
| **全选 / 取消** | 选择或取消选择所有步骤 |
| **配置** | 打开该分析器类型的设置 → 分析器选项卡 |
| **运行 N 个步骤** | 按顺序运行选定步骤 |
| **停止** | 当前步骤完成后取消 |
| **清理** | 删除临时目录和 zip 压缩包 |

## MintPy 结果查看器

`velocity` 步骤完成后可用。在分析器面板中点击**查看结果**。

### 时序文件选择器

列出文件夹中所有可用的 `timeseries*.h5` 文件，按质量排序：

```
timeseries_ERA5_ramp_demErr.h5   ← 最佳（所有校正已应用）
timeseries_ERA5_ramp.h5
timeseries_ERA5_demErr.h5
timeseries_ERA5.h5
timeseriesResidual_ramp.h5
timeseriesResidual.h5
timeseries.h5                    ← 原始（无校正）
```

选择用于像素时序提取的文件，然后点击**绘图**。

### 速度图

- 色标：蓝色 = 沉降，红色 = 抬升，对称范围缩放至第 98 百分位
- 零位移像素为透明
- 不透明度：85%

### 像素时序

点击速度叠加层上的任意点，提取并绘制该像素的位移时序。

- 位移以 **mm** 为单位显示，相对于第一个获取日期（第一个日期 = 0）
- X 轴显示日期
- 图表跟随应用的深色/浅色主题

## 键盘与鼠标参考

| 操作 | 结果 |
|--------|--------|
| 点击地图（矩形框模式，第 1 次） | 设置 AOI 的第一个角 |
| 点击地图（矩形框模式，第 2 次） | 完成 AOI 矩形框 |
| 双击地图（多边形模式） | 关闭多边形 |
| 点击速度叠加层 | 提取像素时序 |
| 点击覆盖范围 | 打开场景详情面板 |
| 滚轮 | 缩放地图 |
| 拖动 | 平移地图 |
